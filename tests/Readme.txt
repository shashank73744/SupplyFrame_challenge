This is a web app which displays the images of dogs and their breed which 
is obtained using the api from "https://dog.ceo/dog-api/"

Steps to run the web app 
1. Unzip the file 
2. Run "npm install" in the command prompt of the root folder which will install all the dependencies
3. Run "npm run start" to run the server
4. In the chrome browser open "https://localhost:8000"

Development
1.Used Express to make Ajax calls
2.Used Ejs as template engine
3.Used nightwatch for testing
4.Used sass for adding styles
5.Used bootstrap and only loaded the required files
